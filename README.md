# Pro Plank Timer

프로 플랭크 타이머는 플랭크 운동을 **간편하게** 도와주는 웹 애플리케이션입니다.

## 주요 기능 (Features)
- **간편한 루틴 가이드**: 준비, 일반 플랭크, 사이드 플랭크(좌/우), 휴식으로 이어지는 루틴을 **쉽게** 따라할 수 있습니다.
- **사용자 설정 (Customizable Settings)**:
  - 총 세트 수 설정
  - 준비 및 자세 변경 시간 설정
  - 플랭크 및 사이드 플랭크 수행 시간 설정
  - 세트 간 휴식 시간 설정
  - 설정값은 브라우저에 자동 저장됩니다.
- **PWA 지원 (Progressive Web App)**: 모바일 및 데스크탑에 앱처럼 설치하여 오프라인에서도 사용할 수 있습니다.
- **직관적인 UI**: 현재 진행 단계, 남은 시간, 다음 단계를 큰 화면으로 명확하게 보여줍니다.

## 사용 방법 (How to Use)
1. 앱을 실행하고 우측 상단의 톱니바퀴(⚙️) 아이콘을 클릭합니다.
2. 본인의 체력에 맞게 세트 수와 각 동작의 시간을 설정하고 '저장'을 누릅니다.
3. 메인 화면의 '운동 시작' 버튼을 누르면 타이머가 시작됩니다.
4. 소리와 화면의 안내에 따라 운동을 진행하세요.

## 기술 스택
- HTML5, CSS3, JavaScript (Vanilla)
- PWA (Service Worker, Manifest)
